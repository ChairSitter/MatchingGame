Color and Word Matching Game

Rules:

This is a game of luck in which you play against the computer to 25 points.
You have 4 boxes and so does the computer.
Each round, the boxes are assigned a randomly generated background color and text color one-by-one.
Points are awarded after each round as follows:

  • Win the Round:
    +1 point to the side with more matches (background color matches text color of a box)
  • Loose straight:
    +3 point bonus if all 4 of your boxes have a color in common (background color OR text)
  • Strict straight:
    +5 point bonus if all 4 boxes have the same background color, or the same text color
  • Four matches:
    +10 point bonus if all 4 boxes contain matches
  • Perfect round:
    +Automatic win if all background and text colors are the same (perfect round)
